About Galex
Galex is a modern, feature-rich music player designed for music lovers. With a sleek interface and powerful features, it offers the ultimate music listening experience. Enjoy your favorite Tamil songs from top artists with crystal clear audio quality.
Features

Save your favorite songs to listen later

Responsive Design
Works perfectly on all devices
Technical Details
Galex is built with modern web technologies including HTML5, CSS3, and JavaScript. It uses the Web Audio API for high-quality audio playback and localStorage for saving user preferences and liked songs. The application is fully responsive and works offline for previously loaded content.
